# Music

Drop a royalty-free MP3 file named `music.mp3` into this folder.

Recommended: Download a calm/confident corporate ambient track from Pixabay:
- Search "calm corporate ambient" at https://pixabay.com/music/search/calm%20corporate/
- Download as MP3
- Rename to `music.mp3`
- Place in this folder

Then uncomment the `<Audio>` line in `src/DonnaDemo.tsx` to enable it.
