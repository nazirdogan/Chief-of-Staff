Option C — The Signal

Donna

An abstract mark: three concentric arcs radiating from a single point — the signal. Always broadcasting, always listening. This is Donna as an intelligence layer, not a person. Pure function.

01 — Logo
The Signal Mark

Three arcs radiate from a central dawn point. Each arc represents a layer: ingestion, analysis, action. The mark works as both a standalone icon and paired with the Cormorant wordmark.

02 — Colour & Type

Same palette and type stack as Options A and B. The Signal uses fewer colours in practice — Midnight, Dawn, and Paper dominate. The arcs use opacity variations of Midnight rather than multiple colours, keeping the mark clean at small sizes.
